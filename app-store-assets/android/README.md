# Google Play Store Assets

## Required Assets

### App Icon
- **Size**: 512x512 pixels
- **Format**: PNG (32-bit with alpha)
- **File**: `app-icon.png`

### Feature Graphic
- **Size**: 1024x500 pixels
- **Format**: PNG or JPG
- **File**: `feature-graphic.png`

### Screenshots
- **Size**: At least 320px wide, max 3840px wide
- **Aspect Ratio**: 16:9 or 9:16
- **Format**: PNG or JPG
- **Required**: At least 2 screenshots
- **Files**: `screenshot-1.png`, `screenshot-2.png`, etc.

### Phone Screenshots (Recommended)
- **Size**: 1080x1920 pixels (9:16)
- **Files**: `phone-screenshot-1.png`, `phone-screenshot-2.png`, etc.
